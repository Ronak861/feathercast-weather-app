# 🌤️ FeatherCast Weather App

A modern and responsive Weather Forecast Web Application built using **HTML, CSS, and JavaScript** with the **OpenWeather API**. The application provides real-time weather information, a 5-day forecast, geolocation support, and recent search history in a clean and user-friendly interface.

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Automatic weather based on current location
- 🌡️ Real-time temperature and weather conditions
- 💧 Humidity information
- 💨 Wind speed details
- 🤗 Feels Like temperature
- 📅 5-Day Weather Forecast
- 🕒 Recent Search History
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Ronak861/feathercast-weather-app.git

```

2. Open the project folder.

3. Open `index.html` in your browser or run it using **Live Server** in Visual Studio Code.
